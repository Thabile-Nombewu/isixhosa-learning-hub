Glossary / Igalelo lamagama

Key Python terms in English and isiXhosa

Understand the language of code / Qonda ulwimi lwekhodi

This glossary helps you learn Python vocabulary.
Le galelo lamagama likunceda ufunde amagama ePython.

Variable / Inguqulelo

English: A container for storing data.
isiXhosa: Isitya esigcina ulwazi.

Loop / Iiloop

English: Repeats actions multiple times.
isiXhosa: Iphinda izinto amaxesha amaninzi.

Function / Umsebenzi

English: A block of code that performs a task.
isiXhosa: Iqhekeza lekhodi elenza umsebenzi othile.

Condition / Imeko

English: A rule that decides what happens next.
isiXhosa: Umthetho ogqiba ukuba kwenzeke ntoni.
