Python Tutorials / Izifundo zePython

Learn coding step-by-step in isiXhosa and English

Start Learning / Qalisa ukufunda

These tutorials will help you understand Python basics.
Ezi zifundo ziza kukunceda uqonde izinto ezisisiseko zePython.

Loops / Iiloop

English: Loops repeat actions.
isiXhosa: Iiloop ziphinda izinto amaxesha amaninzi. 

for i in range(3):
    print("Molo!")

Variables / Iinguqulelo

English: Variables store information.
isiXhosa: Iinguqulelo zigcina ulwazi. 

name = "Wonga"
print(name)

Conditionals / Iimeko

English: Conditionals make decisions.
isiXhosa: Iimeko zenze izigqibo. 

if age > 18:
    print("Wamkelekile!")
